# Awesome Antigravity CLI

A curated list of awesome tools, skills, integrations, and resources for [Antigravity CLI](https://antigravity.google) (`agy`), Google's agent-first coding CLI.

Antigravity CLI shares the same agent harness as Antigravity 2.0 and supports Agent Skills, Hooks, Subagents, and Extensions. New to `agy`? Start with the [official CLI documentation](https://antigravity.google/docs/cli).

## Official Resources

- [Antigravity CLI](https://github.com/google-antigravity/antigravity-cli) - The official CLI repository: reasoning, execution, and orchestration from the Antigravity agent harness in your terminal.
- [Antigravity SDK for Python](https://github.com/google-antigravity/antigravity-sdk-python) - Official Python library for building agents on the Antigravity platform.

## Skills & Agents

- [antigravity-skills](https://github.com/rmyndharis/antigravity-skills) - A curated collection of Agent Skills for Google Antigravity.
- [Sample Antigravity Skills](https://github.com/rominirani/antigravity-skills) - Sample skills from Google Cloud DevRel, from git workflows to release notes.
- [Agent Skills Registry](https://github.com/tech-leads-club/agent-skills) - Secure, validated skill registry for professional AI coding agents, with Antigravity support.
- [agentic-awesome-skills](https://github.com/sickn33/agentic-awesome-skills) - Installable library of 1,900+ agentic skills with an installer CLI, supporting Antigravity.

## Google-Built Skills & Extensions

- [Conductor](https://github.com/gemini-cli-extensions/conductor) - Spec-Driven Development plugin for AI coding agents including Antigravity: specify, plan, and implement.
- [Mantis](https://github.com/google/mantis) - Modular, stack-agnostic toolkit of security review skills for AI coding agents.
- [Modern Web Guidance](https://github.com/GoogleChrome/modern-web-guidance) - Web development guidance skills from the Chrome team, installable via `agy`.
- [Data Agent Kit](https://github.com/GoogleCloudPlatform/data-agent-kit) - Google Cloud's kit for data agents, installable via `agy`.
- [BigQuery Data Analytics](https://github.com/gemini-cli-extensions/bigquery-data-analytics) - Official data analytics skills for BigQuery.
- [Looker Skills](https://github.com/gemini-cli-extensions/looker) - Official skills for Looker.
- [Cloud SQL for PostgreSQL](https://github.com/gemini-cli-extensions/cloud-sql-postgresql) - Official skills for Cloud SQL for PostgreSQL.

## Integrations

- [antigravity-for-claude-code](https://github.com/yuting0624/antigravity-for-claude-code) - Claude Code plugin that runs Antigravity CLI as a collaborating sub-agent with model routing across the SDLC.
- [claude-antigravity-agents](https://github.com/markfulton/claude-antigravity-agents) - Claude Code skill that delegates coding, review, analysis, and research jobs to `agy` sub-agents.

## Orchestration & Session Management

- [oh-my-antigravity](https://github.com/Joonghyun-Lee-Frieren/oh-my-antigravity) - Context-engineering-powered multi-agent team workflow pack for Antigravity CLI.
- [antigravity-cloud-run](https://github.com/ykdojo/antigravity-cloud-run) - Run multiple `agy` sessions in isolated Docker containers, locally or on Cloud Run, managed from a web dashboard.

## Utilities

- [antigravity-cli-statusline](https://github.com/AndyAWD/antigravity-cli-statusline) - Cross-platform statusline skill showing API quota, token usage, Git branch, and system resources.
